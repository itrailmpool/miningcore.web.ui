Miner Configuration

Getting started

To get started mining and use this pool you need the following

  • {{ coinName }} Wallet address
  • Crypto mining software that supports the {{ coinName }} coin and aglorithm {{ aglorithm }}
  • Hardware to run it on (ASIC miner)

{{ coinName }} Wallet address

A wallet address is needed to payout you shares mined at this pool server.
When the total mined value is past the payout threshold, we will send your coin to this wallet address.


Crypto mining software

To mine at this pool you can use any miner supporting the {{ aglorithm }} aglorithm or {{ coinName }} coin.
Use an search engine and search for "{{ coinName }} miner software". download the miner software and configure your crypto miner.

Where:

  • POOL STRATUM ADDRESS AND PORT - one off the stratum addresses above in the Pool Configuration section
  • YOUR_WALLET_ADDRESS.WORKERNAME - your {{ coinName }} wallet address and WORKERNAME split by '.'
  • WORKERNAME - an optional workername can be used to identify the miner or RIG
  • PASSWORD - use 'x' or leave it blank
